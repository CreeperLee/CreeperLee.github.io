CentOS 7 安装 JDK8

发表于 2020-11-07

CentOS版本：CentOS Linux release 7.5.1804 (Core)

JDK版本：jdk-8u181-linux-x64

1、先去java官网下载jdk（注意32位还是64位）

2、执行 rpm -ivh jdk-8u181-linux-x64.rpm 命令安装jdk

3、执行 java -version 命令检查是否安装成功
